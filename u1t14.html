Imprimir Datos en Pantalla

Descripción:
En Java, la función System.out.println() se utiliza para mostrar información en la consola. Es una herramienta útil para verificar resultados, mostrar mensajes, y depurar el código.
Ejemplo:
public class EjemploImpresion {
public static void main(String[] args) {
System.out.println("Hola, este es un mensaje en la consola.");
}
}
Este código imprimirá:
Hola, este es un mensaje en la consola.
