Formulario: Ciclo `for` en Java


a) Declara la variable del ciclo.
b) Incrementa la variable del ciclo.
c) Evalúa si el ciclo debe continuar ejecutándose.


a) El ciclo termina.
b) El ciclo se ejecuta una vez más.
c) Se genera un error.


a) `for (int i = 10; i < 1; i--)`
b) `for (int i = 10; i >= 1; i--)`
c) `for (int i = 10; i <= 1; i--)`
