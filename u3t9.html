Problema 1: Suma de Elementos en un Arreglo Bidimensional

Descripci√≥n:
Dado un arreglo bidimensional, suma todos los elementos y muestra el resultado. Ejemplo:

public class SumaMatriz {
public static void main(String[] args) {
int[][] matriz = {
{1, 2, 3},
{4, 5, 6},
{7, 8, 9}
};
int suma = 0;

for (int i = 0; i < matriz.length; i++) {
for (int j = 0; j < matriz[i].length; j++) {
suma += matriz[i][j];
}
}

System.out.println("La suma de todos los elementos de la matriz es: " + suma);
}
}
