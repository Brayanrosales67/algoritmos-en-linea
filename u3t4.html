Algoritmo de Busqueda Secuencial

Descripción:
La búsqueda secuencial revisa cada elemento de un arreglo uno por uno hasta encontrar el valor buscado o hasta llegar al final del arreglo.

Ejemplo:
public class BusquedaSecuencial {
public static void main(String[] args) {
int[] numeros = {3, 7, 1, 14, 9};
int buscado = 14;
boolean encontrado = false;

for (int i = 0; i < numeros.length; i++) {
if (numeros[i] == buscado) {
System.out.println("Número encontrado en la posición: " + i);
encontrado = true;
break;
}
}

if (!encontrado) {
System.out.println("Número no encontrado en el arreglo.");
}
}
}
