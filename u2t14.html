Problema General 1: Cálculo de Promedio y Clasificación de Calificación

Descripción:
Crea un programa que pida al usuario tres calificaciones, calcule el promedio e indique la clasificación de la calificación promedio según estas reglas:
Aprobado si el promedio es mayor o igual a 60
Reprobado si el promedio es menor a 60
Ejemplo:
import java.util.Scanner;

public class PromedioCalificacion {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
int calificacion1, calificacion2, calificacion3;

System.out.print("Ingrese la primera calificación: ");
calificacion1 = scanner.nextInt();

System.out.print("Ingrese la segunda calificación: ");
calificacion2 = scanner.nextInt();

System.out.print("Ingrese la tercera calificación: ");
calificacion3 = scanner.nextInt();

double promedio = (calificacion1 + calificacion2 + calificacion3) / 3.0;

System.out.println("El promedio es: " + promedio);

if (promedio >= 60) {
System.out.println("Estado: Aprobado");
} else {
System.out.println("Estado: Reprobado");
}
scanner.close();
}
}
