Algoritmo de Ordenamiento Burbuja

Descripción:
l ordenamiento burbuja recorre repetidamente el arreglo y compara elementos adyacentes, intercambiándolos si están en el orden incorrecto.
Es sencillo pero ineficiente para arreglos grandes.

Ejemplo:

public class OrdenamientoBurbuja {
public static void main(String[] args) {
int[] numeros = {5, 3, 8, 1, 2};

for (int i = 0; i < numeros.length - 1; i++) {
for (int j = 0; j < numeros.length - i - 1; j++) {
if (numeros[j] > numeros[j + 1]) {
int temp = numeros[j];
numeros[j] = numeros[j + 1];
numeros[j + 1] = temp;
}
}
}

System.out.print("Arreglo ordenado: ");
for (int numero : numeros) {
System.out.print(numero + " ");
}
}
}
