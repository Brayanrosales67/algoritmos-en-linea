Arreglo Unidimensional

Descripción:
Un arreglo unidimensional es una estructura de datos que almacena una secuencia de elementos del mismo tipo en una sola línea, accesibles mediante un índice.
Permite almacenar y manipular múltiples valores en una sola variable.

Ejemplo:
public class ArregloUnidimensional {
public static void main(String[] args) {
int[] numeros = {10, 20, 30, 40, 50};

for (int i = 0; i < numeros.length; i++) {
System.out.println("Elemento en la posición " + i + ": " + numeros[i]);
}
}
}
