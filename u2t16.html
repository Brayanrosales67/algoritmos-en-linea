Problema General 3: Tabla de Multiplicar con Opciones

Descripción:
Este programa permite al usuario ingresar un número y una opción para mostrar la tabla de multiplicar del 1 al 10 usando un bucle for, while o do-while.
El usuario elige la estructura repetitiva a usar.

Ejemplo:
import java.util.Scanner;

public class TablaMultiplicar {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);

System.out.print("Ingrese un número para ver su tabla de multiplicar: ");
int numero = scanner.nextInt();

System.out.println("Seleccione la estructura de repetición:");
System.out.println("1. for");
System.out.println("2. while");
System.out.println("3. do-while");
int opcion = scanner.nextInt();

switch (opcion) {
case 1:
System.out.println("Tabla de multiplicar con for:");
for (int i = 1; i <= 10; i++) {
System.out.println(numero + " x " + i + " = " + (numero * i));
}
break;

case 2:
System.out.println("Tabla de multiplicar con while:");
int j = 1;
while (j <= 10) {
System.out.println(numero + " x " + j + " = " + (numero * j));
j++;
}
break;

case 3:
System.out.println("Tabla de multiplicar con do-while:");
int k = 1;
do {
System.out.println(numero + " x " + k + " = " + (numero * k));
k++;
} while (k <= 10);
break;

default:
System.out.println("Opción no válida.");
}
scanner.close();
}
}
