Raiz Cuadrada Metodo sqrt

Descripción:
Para calcular la raíz cuadrada en Java, utilizamos el método Math.sqrt. Este método toma un número como parámetro y devuelve su raíz cuadrada. Es parte de la clase Math, que proporciona herramientas útiles para realizar cálculos matemáticos. El parámetro que se pasa al método debe ser un valor de tipo double, y el resultado devuelto también será de tipo double, lo que permite realizar cálculos precisos incluso con números decimales.
Ejemplo:
public class EjemploRaizCuadrada {
public static void main(String[] args) {
double numero = 16;
double raizCuadrada = Math.sqrt(numero);
System.out.println("La raíz cuadrada de 16 es: " + raizCuadrada);
}
}
Resultado en la consola:
La raíz cuadrada de 16 es: 4.0
