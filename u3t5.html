Algoritmo de Busqueda Binaria

Descripción:
La búsqueda binaria requiere un arreglo ordenado y divide el rango de búsqueda a la mitad en cada paso, reduciendo el tiempo de búsqueda en comparación con la búsqueda secuencial.

Ejemplo:

import java.util.Arrays;

public class BusquedaBinaria {
public static void main(String[] args) {
int[] numeros = {1, 3, 7, 9, 14};
int buscado = 7;

Arrays.sort(numeros); // Asegurarse de que el arreglo esté ordenado
int posicion = Arrays.binarySearch(numeros, buscado);

if (posicion >= 0) {
System.out.println("Número encontrado en la posición: " + posicion);
} else {
System.out.println("Número no encontrado en el arreglo.");
}
}
}
