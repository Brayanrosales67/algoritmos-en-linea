Estructura Repetitiva do whiel

Descripción:
El bucle do-while es una estructura de control en Java que asegura que el bloque de código se ejecute al menos una vez, independientemente de la condición. Esto se debe a que la verificación de la condición se realiza al final de cada iteración, y no al inicio como en otros bucles como while o for.
Sintaxis del do-while:
do {
// Bloque de código a ejecutar
} while (condición);
Bloque de código: Se ejecuta al menos una vez antes de evaluar la condición.
Condición: Una expresión booleana que determina si el ciclo continuará ejecutándose. Si es true, se repite el bloque. Si es false, el ciclo se detiene.

Ejemplo:
import java.util.Scanner;

public class DoWhileEjemplo {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
int numero;
int suma = 0;

do {
System.out.print("Ingrese un número (negativo para salir): ");
numero = scanner.nextInt();
if (numero >= 0) {
suma += numero;
}
} while (numero >= 0);

System.out.println("La suma de los números ingresados es: " + suma);
scanner.close();
}
}
