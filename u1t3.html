Algoritmo Cuantitativo

Descripción:
Un algoritmo cuantitativo es un conjunto de pasos bien definidos que se emplea para procesar datos numéricos y obtener resultados cuantificables. A diferencia de los algoritmos cualitativos, que se centran en categorías o descripciones no numéricas, los algoritmos cuantitativos operan con valores específicos, como números, medidas o estadísticas.
Ejemplo:
Calcular el área de un rectángulo:
1.Medir la base y la altura del rectángulo.
2.Multiplicar la base por la altura.
3.El resultado es el área del rectángulo.
