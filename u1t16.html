Potencia Metodo Pow

Descripción:
La clase Math en Java proporciona el método pow para calcular potencias. Math.pow(base, exponente) devuelve el resultado de elevar la base al exponente. Este método es útil para realizar cálculos matemáticos más avanzados, como el cálculo de áreas, volúmenes o cualquier operación que requiera exponenciación. El método acepta dos parámetros de tipo double y también devuelve un resultado de tipo double, lo que garantiza precisión en operaciones con números decimales o muy grandes. Ejemplo:
public class EjemploPotencia {
public static void main(String[] args) {
double base = 3;
double exponente = 4;
double resultado = Math.pow(base, exponente);
System.out.println("3 elevado a la 4 es: " + resultado);
}
}
Resultado en la consola:
3 elevado a la 4 es: 81.0
