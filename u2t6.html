Estructura Repetitiva for

Descripción:
La estructura for es uno de los ciclos más utilizados en Java. Sirve para repetir un bloque de código un número específico de veces o hasta que se cumpla una condición determinada. Se utiliza especialmente cuando sabemos de antemano cuántas veces necesitamos que el ciclo se ejecute.
Sintaxis básica del ciclo for: java
Copiar código
for (inicialización; condición; actualización) {
// Bloque de código a ejecutar
Inicialización: Es la parte donde se declara y/o inicializa una variable de control. Se ejecuta solo una vez al inicio del ciclo.
Condición: Es una expresión booleana que se evalúa antes de cada iteración. Si es true, el bloque de código se ejecuta; si es false, el ciclo termina.
Actualización: Es la operación que se ejecuta al final de cada iteración, como incrementar o decrementar la variable de control.
}

Ejemplo:
public class EstructuraFor {
public static void main(String[] args) {
for (int i = 1; i <= 5; i++) {
System.out.println("Iteración: " + i);
}
}
}
