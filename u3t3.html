Problema 2: Encontrar el Mayor y el Menor en un Arreglo Unidimensional

Descripción:
Dado un arreglo de enteros, encuentra y muestra el valor máximo y el valor mínimo.

Ejemplo:
public class MayorMenorArreglo {
public static void main(String[] args) {
int[] numeros = {8, 3, 15, 7, 12};
int maximo = numeros[0];
int minimo = numeros[0];

for (int numero : numeros) {
if (numero > maximo) {
maximo = numero;
}
if (numero < minimo) {
minimo = numero;
}
}

System.out.println("El valor máximo es: " + maximo);
System.out.println("El valor mínimo es: " + minimo);
}
}
