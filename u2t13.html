Problema 7 secuencial con estructura selectiva

Descripción:
Sumar hasta encontrar un negativo. Este problema requiere que se lean números ingresados por el usuario hasta que se ingrese un número negativo.
Al final, el programa debe mostrar la suma total de los números positivos.
Ejemplo:
import java.util.Scanner;

public class SumaHastaNegativo {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
int numero;
int suma = 0;

System.out.println("Ingrese números positivos para sumar. Ingrese un número negativo para terminar.");

do {
System.out.print("Número: ");
numero = scanner.nextInt();
if (numero >= 0) {
suma += numero;
}
} while (numero >= 0);

System.out.println("La suma total de los números positivos es: " + suma);
scanner.close();
}
}
