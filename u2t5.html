Estructuras Selectivas switch

Descripción:
La estructura switch en Java es una herramienta fundamental que permite controlar el flujo de un programa basándose en el valor de una sola variable o expresión. A diferencia de las estructuras if-else, que evalúan condiciones booleanas, switch compara el valor de una variable con múltiples opciones posibles, conocidas como "casos". Esta estructura resulta especialmente útil cuando se trabaja con múltiples escenarios para un solo valor, como menús de opciones o valores categóricos.
Sintaxis básica de switch
switch (expresión) {
case valor1:
// Código a ejecutar si expresión == valor1
break; // Termina el caso
case valor2:
// Código a ejecutar si expresión == valor2
break;
// Puedes añadir tantos casos como necesites
default:
// Código a ejecutar si ningún caso coincide
}
Expresión: La variable o expresión a evaluar debe ser de un tipo permitido, como:
int, short, byte, char
String
Enums
Casos (case): Cada caso evalúa un posible valor de la expresión. Si coincide, se ejecuta el bloque de código asociado.
break: Se utiliza para evitar que la ejecución continúe en los casos siguientes. Si se omite, se produce un fenómeno conocido como fallthrough.
default: Este bloque es opcional, pero se ejecuta si ninguno de los casos coincide.
Ejemplo:
iimport java.util.Scanner;

public class EstructuraSwitch {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
System.out.print("Ingrese un día de la semana (1-7): ");
int dia = scanner.nextInt();

switch (dia) {
case 1: System.out.println("Lunes"); break;
case 2: System.out.println("Martes"); break;
case 3: System.out.println("Miércoles"); break;
case 4: System.out.println("Jueves"); break;
case 5: System.out.println("Viernes"); break;
case 6: System.out.println("Sábado"); break;
case 7: System.out.println("Domingo"); break;
default: System.out.println("Día inválido.");
}
scanner.close();
}
}
