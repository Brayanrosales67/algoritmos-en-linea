Algoritmos Secuenciales

Descripción:
En Java, estos algoritmos se implementan utilizando un conjunto de sentencias que se ejecutan en el mismo orden en que se escriben en el programa.
Por ejemplo, el cálculo del área de un rectángulo sigue un algoritmo secuencial, donde primero se leen las dimensiones, luego se calcula el área
multiplicando la base por la altura, y finalmente se muestra el resultado en pantalla.
Ejemplo:
Algoritmo para calcular el promedio de tres números:
1.Tomar tres números.
2.Sumar los tres números.
3.Dividir la suma entre 3.
4.Mostrar el promedio.
