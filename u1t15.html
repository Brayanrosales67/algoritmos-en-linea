Concatenar

Descripción:
La concatenación en Java se utiliza para unir dos o más cadenas de texto. El operador + permite combinar cadenas o variables con valores para formar un mensaje completo. Esta técnica es muy útil para construir salidas dinámicas o mensajes personalizados, integrando texto literal con valores obtenidos de variables.
Ejemplo:
public class EjemploConcatenacion {
public static void main(String[] args) {
String nombre = "Juan";
int edad = 25;
System.out.println("Mi nombre es " + nombre + " y tengo " + edad + " años.");
}
}
Resultado en la consola:
Mi nombre es Juan y tengo 25 años.
