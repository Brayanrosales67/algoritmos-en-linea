Numero Aleatorio Metodo Random

Descripción:
Para generar números aleatorios en Java, usamos el método Math.random(), que devuelve un número decimal entre 0.0 y 1.0 (excluyendo el 1.0). Este método es útil cuando necesitamos generar valores aleatorios dentro de un rango específico.
Al multiplicar el resultado de Math.random() por un valor determinado y luego redondearlo o ajustarlo, podemos obtener números en el rango deseado.
Ejemplo:
public class EjemploNumeroAleatorio {
public static void main(String[] args) {
int min = 1;
int max = 10;
int numeroAleatorio = (int)(Math.random() * (max - min + 1) + min);
System.out.println("Número aleatorio entre 1 y 10: " + numeroAleatorio);
}
}
Este código imprimirá un número aleatorio entre 1 y 10, como:
Número aleatorio entre 1 y 10: 7
