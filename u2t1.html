Pedir Datos del Teclado

Descripción:
Para recibir datos del usuario en Java, usamos la clase Scanner, que permite leer diferentes tipos de datos. Para poder hacer uso de el debemos importar
la libreria java.util de la siguiente forma: import java.util.Scanner;
Ejemplo:
import java.util.Scanner;

public class PedirDatos {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
System.out.print("Ingrese su nombre: ");
String nombre = scanner.nextLine();
System.out.println("Hola, " + nombre + "!");
scanner.close();
}
}
Este programa pedirá al usuario su nombre y luego lo mostrará en pantalla.
