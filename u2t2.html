Estructuras Selectivas if

Descripción:
En Java, la estructura if se utiliza para tomar decisiones en el código. Evalúa una condición, y si la condición es verdadera, ejecuta un bloque de código. Si la condición no se cumple, se puede usar un bloque else para ejecutar otras instrucciones.
La estructura básica es:
java
Copiar código
if (condición) {
// Bloque de código si la condición es verdadera
} else { // Bloque de código si la condición es falsa }

Ejemplo:
import java.util.Scanner;

public class EstructuraIf {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
System.out.print("Ingrese un número: ");
int numero = scanner.nextInt();

if (numero > 0) {
System.out.println("El número es positivo.");
} else {
System.out.println("El número es negativo o cero.");
}
scanner.close();
}
}
Este programa pedirá al usuario un numero, y lo comparara para ver si es mayor a 0, y ejecutara una orden en caso de cumplirse la condicion.
En caso de que no se cumpla ejecutara lo que esta dentro de la estructura de else.
